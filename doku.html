📘 Webspell-RM 3.0 Dokumentation

Einsteigerhilfe, Plugin-Erweiterungen und Entwicklertipps auf einer Seite.

🚀 Erste Schritte

  • Webserver mit PHP 8.1+ und MySQL/MariaDB
  • mod_rewrite aktiviert (für sprechende URLs)
  • Verzeichnisse /includes, /templates_c, /images beschreibbar

🛠 Installation

  1. Repository von GitHub klonen oder ZIP herunterladen
  2. Projekt auf den Webserver hochladen
  3. /install im Browser öffnen und Setup ausführen
  4. Datenbank & Admin-Zugang einrichten
  5. Fertig! Du kannst dich im /admincenter.php anmelden

🧩 Plugin-Installation

Ab Webspell-RM 3.0 erfolgt die Plugin-Verwaltung über den Plugin-Installer:

  1. Repository von GitHub klonen oder ZIP-Archiv herunterladen
  2. Projektverzeichnis auf deinen Webserver hochladen
  3. Im Browser /install aufrufen und das Setup ausführen
  4. Datenbankverbindung und Admin-Zugang einrichten
  5. Fertig! Du kannst dich im Adminbereich unter /admincenter.php anmelden
  6. Plugins kannst du bequem über den integrierten Plugin-Installer verwalten und aktualisieren

📦 Wichtige Plugins

Die folgende Liste gibt einen Überblick über die wichtigsten mitgelieferten Plugins von Webspell-RM. Jedes Plugin erweitert das System um spezifische Funktionen und kann über das Admin-Panel konfiguriert werden.

  • 📰 News: Ein flexibles News-System zur Veröffentlichung von Nachrichten und Ankündigungen. Unterstützt Kategorien, Bilder, Archivierung, mehrsprachige Inhalte sowie ein Veröffentlichungsdatum. Inklusive Admin-Modul mit Editor und Kommentaroption.
  • 🖼️ Gallery: Verwaltung und Darstellung von Bildern in einer dynamischen Galerie. Funktionen: Drag-&-Drop-Sortierung, mehrseitige Verwaltung, Bild-Upload mit Vorschau, Klassensystem (.wide/.tall/.big), sowie anpassbare Galerie-Layouts.
  • 📈 Counter: Ein datenschutzfreundlicher Besucherzähler mit Session-Schutz und Bot-Erkennung. Zeichnet echte Besucher anonymisiert auf, bietet Zeitfilterung (Tag/Woche/Monat), einfache Diagramme und CSV-Export. Nur im Adminbereich sichtbar.
  • 🤝 Partners / Sponsors: Anzeige und Verwaltung von Partner- oder Sponsorenlogos mit optionalem Link. Logos können hochgeladen, sortiert und mit Ziel-URLs versehen werden. Ideal für Unternehmenskooperationen oder Werbepartner.
  • ✍️ Articles: Redaktionssystem zur Veröffentlichung umfangreicher Artikel und Blogbeiträge. Enthält Bewertungssystem (1–5 Sterne), Kommentarfunktion mit Bearbeitung, Kategorieverwaltung und mehrsprachige Unterstützung.
  • 📄 About / Resume / Pricing: Drei kombinierte Info-Module für Unternehmenspräsentation.
    • About: Erzählt die Geschichte und Philosophie des Projekts.
    • Resume: Lebenslauf-ähnliche Darstellung z. B. für Freelancer oder Teams.
    • Pricing: Darstellung von Preisplänen mit Features, ideal für Services.

📦 Important Plugins

The following list provides an overview of the most important built-in plugins of Webspell-RM. Each plugin adds specific functionality and can be configured via the admin panel.

  • 📰 News: A flexible news system for publishing announcements and updates. Supports categories, images, archiving, multilingual content, and publish dates. Includes admin module with editor and optional comments.
  • 🖼️ Gallery: Manage and display images in a dynamic gallery layout. Features: drag-and-drop sorting, multi-page navigation, image uploads with preview, class system (.wide/.tall/.big), and customizable layouts.
  • 📈 Counter: A GDPR-friendly visitor counter with session protection and bot detection. Records real visitors anonymously, provides time filters (day/week/month), basic charts, and CSV export. Visible to admins only.
  • 🤝 Partners / Sponsors: Display and manage partner or sponsor logos with optional links. Upload, sort and assign URLs to each logo – ideal for business cooperation or advertising banners.
  • ✍️ Articles: Editorial system for publishing detailed articles and blog posts. Includes star rating system (1–5), comment function with editing, category management and multilingual support.
  • 📄 About / Resume / Pricing: Three combined info modules for presenting your organization or services.
    • About: Tells the story and philosophy of your project.
    • Resume: CV-style overview, useful for freelancers or teams.
    • Pricing: Displays price plans with features – ideal for commercial offerings.

📦 Plugin Importanti

L’elenco seguente presenta una panoramica dei principali plugin inclusi in Webspell-RM. Ogni plugin estende il sistema con funzionalità specifiche ed è configurabile dal pannello di amministrazione.

  • 📰 News: Un sistema di notizie flessibile per pubblicare annunci e aggiornamenti. Supporta categorie, immagini, archiviazione, contenuti multilingue e data di pubblicazione. Include modulo admin con editor e opzione commenti.
  • 🖼️ Gallery: Gestione e visualizzazione di immagini in una galleria dinamica. Funzioni: ordinamento tramite drag & drop, gestione multi-pagina, caricamento immagini con anteprima, sistema classi (.wide/.tall/.big) e layout personalizzabili.
  • 📈 Counter: Contatore visite conforme al GDPR, con protezione sessioni e filtro bot. Registra visitatori reali in modo anonimo, filtri per periodo (giorno/settimana/mese), grafici semplici e esportazione CSV. Visibile solo nell'area admin.
  • 🤝 Partners / Sponsors: Visualizzazione e gestione dei loghi di partner o sponsor con link opzionali. Carica loghi, ordina e assegna URL di destinazione. Ideale per collaborazioni aziendali o spazi pubblicitari.
  • ✍️ Articles: Sistema editoriale per pubblicare articoli e post dettagliati. Include sistema di valutazione a stelle (1–5), commenti con modifica, gestione categorie e supporto multilingua.
  • 📄 About / Resume / Pricing: Tre moduli informativi per presentare il tuo progetto o servizio.
    • About: Racconta la storia e la filosofia del progetto.
    • Resume: Presentazione in stile CV, utile per freelance o team.
    • Pricing: Visualizzazione dei piani tariffari con funzionalità – ideale per offerte commerciali.
